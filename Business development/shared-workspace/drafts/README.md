# Drafts Workspace Contract

This directory holds the tracked planning-cascade artifacts that sit between strategy and execution.

## Required Files

- `strategy-brief-YYYY-MM-DD.md`
  - Produced by BB during the Tuesday strategy refresh
  - Summarizes active priorities, campaign themes, blockers, and required downstream actions
- `weekly-plan-YYYY-MM-DD.md`
  - Produced by DPM during the Thursday weekly planning cycle
  - Converts current strategy into a dated campaign plan ready for PA review

## Operating Rules

- Files in this directory are shared governance artifacts, not personal working notes.
- Each artifact must include a `Sources Consulted` section.
- Each artifact should link to the upstream and downstream VITA issues it depends on.
- If the latest expected artifact is missing, PA should flag the planning cascade as degraded in the next board digest.
