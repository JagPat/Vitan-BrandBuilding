# Email Templates

Standardized HTML templates for Vitan Architects outreach and communications.

## Templates
- **vitan-modern-outreach.html**: High-polish template for external prospects, award bodies, and media.
- **action-required.html**: System-level internal template for agent-to-agent or agent-to-board alerts.
- **board-digest.html**: Weekly/Daily summary template for Board reviews.

## Usage
Use `scripts/generate_branded_email.py` or equivalent to populate the placeholders (e.g., {{SUBJECT}}, {{CONTENT_HTML}}).

## Brand Rules
- Always use the canonical domain: **vitan.in**.
- Always include the tagline: **Adding life, every square foot**.
- Always use the official primary red: **#D42B2B**.
