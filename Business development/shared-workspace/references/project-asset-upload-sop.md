# Project Asset Upload SOP

Owner: Founding Engineer (FE)  
Effective date: 2026-04-22

## Purpose

Ensure every project targeted for awards/editorials has complete assets at least 30 days before deadline.

## Required Paths

- Project photos: `Business development/PHOTOS FOR SAMPLE PROJECT/{PROJECT_FOLDER}/`
- Asset checklist: `Business development/PHOTOS FOR SAMPLE PROJECT/{PROJECT_FOLDER}/ASSET_CHECKLIST.md`
- Submission calendar: `Business development/shared-workspace/references/submission-calendar.md`

## Minimum Checklist (per project)

`ASSET_CHECKLIST.md` must include:

1. Minimum 8 high-resolution photos (`.jpg/.jpeg/.png/.webp`)
2. Hero image identified
3. Project metadata: location, typology, area, completion year
4. Credits confirmed (architect/photographer/collaborators)
5. Drawing package availability status (plans/sections/elevations)
6. Copyright/usage clearance status

## Workflow

1. BB/BS/OC update `submission-calendar.md` when a new target submission is added.
2. FE monthly routine checks submissions due within 60 days.
3. FE posts readiness table to current BB standup issue.
4. If gaps are found, BB creates follow-up issue(s) for board/owners to upload missing assets.

## Notes

- Project folder names in `submission-calendar.md` must exactly match folder names under `PHOTOS FOR SAMPLE PROJECT`.
- Keep dates in ISO format (`YYYY-MM-DD`) so automation can parse deadlines reliably.
